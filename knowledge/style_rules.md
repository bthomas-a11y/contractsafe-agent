# ContractSafe Non-Negotiable Style Rules

These are applied on the FIRST DRAFT. Every rule is pass/fail.

1. **No em dashes.** Not -- not -. Use commas, periods, parentheses, or restructure.
2. **Paragraphs under 42 words.** Count them. Split if over.
3. **Curly quotes and apostrophes only.** Use actual curly quote characters, never straight quotes.
4. **Links for every tool/competitor mentioned.** If you name a product, link to it.
5. **No "Definitions at a Glance" sections** between TL;DRs and explanatory content.
6. **Landing page copy = headline + 1-2 line subhead + CTA.** Not paragraphs.
7. **No literal unicode escape sequences** in output text. Actual characters only.

## Standard Deliverable Package (every article)
1. Rewritten/written article as markdown
2. SEO meta description (150-160 characters)
3. LinkedIn post (URL in first 3 lines, emojis, visual formatting)
4. X/Twitter post (under 280 chars, link near top)
