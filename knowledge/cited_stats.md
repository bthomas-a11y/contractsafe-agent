# Previously Cited Statistics

Stats that have already been used in published articles. The researcher should
still surface these (they're real data) but should also push harder to find
NEW stats that haven't been used before. An article that only contains
previously-cited stats is lazy research.

## How This File Works
- After a pipeline run passes, the stats used in the article get appended here
- Agent 2 (Subject Researcher) reads this file and deprioritizes stats it finds here
- The researcher should still include 1-2 familiar stats if they're the best fit,
  but the majority of stats in any article should be NEW
